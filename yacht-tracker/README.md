
# Yacht Price Tracker

This repo is a dashboard tool (index) to display a daily crawl of ApooloDuck boat sales to aggregate for sales.
The purpose is to track sales (ask) prices for 38-42 foot monohulls over time. This occasionally collects cats
where they are listed wrongly. Daily crawl of yacht aggregators for 38–43 ft sailboats for sale in Greece, Croatia and Turkey.
Uses GitHub Actions to achieve this. Dashboard is via Git Hub Pages, hense the public nature of the repo.

Appends snapshot rows to `data/listings.csv` — one row per listing per day —
building a longitudinal price dataset.

Sources checked against robots.txt and Apollo selected for the purposes
